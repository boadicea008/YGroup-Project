# BIXI-reports

BIXI-reports is a Python packet for generating predefine reports regarding the data available at https://www.kaggle.com/aubertsigouin/biximtl 

## Requirements

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install python3
pip install pandas
pip install matplotlib
```

Download all files from https://www.kaggle.com/aubertsigouin/biximtl
Unzip the downloaded file
Place files unziped at folder YGroup-Project/data



## Usage (from inside the folder YGroup-Project)

```bash
python .\bixi-reports.py
```

Follow the screen instruction to generate a predefined report.
